storyteller:report

Note:This goal should be used as a Maven report.

Full name:

org.highsource.storyteller:maven-storyteller-plugin:0.2-SNAPSHOT:report

Description:

(no description)

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: site.

Optional Parameters

Name Type Since Description
batikHints Map - Hints to pass to Batik when rendering.
graphVizDotFile String - The plugin uses GraphViz package to render graphs in formats like PDF and so on. If the dot executable is not in PATH, it can be specified manually here.
Default value is: dot.
useBatik boolean - Use Batik to render PNGs instead of Graphviz's default renderer.
Default value is: false.

Parameter Details

batikHints:

Hints to pass to Batik when rendering.
  • Type: java.util.Map
  • Required: No

graphVizDotFile:

The plugin uses GraphViz package to render graphs in formats like PDF and so on. If the dot executable is not in PATH, it can be specified manually here.
  • Type: java.lang.String
  • Required: No
  • Expression: ${graphViz.dotFile}
  • Default: dot

useBatik:

Use Batik to render PNGs instead of Graphviz's default renderer.
  • Type: boolean
  • Required: No
  • Expression: ${useBatik}
  • Default: false
