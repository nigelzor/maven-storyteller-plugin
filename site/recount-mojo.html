storyteller:recount

Full name:

org.highsource.storyteller:maven-storyteller-plugin:0.1-SNAPSHOT:recount

Description:

(no description)

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: verify.

Optional Parameters

Name Type Since Description
archiveDependencyGraphFile File - If specified, the plugin renders archive dependency graph into this file.
Default value is: target/storyteller/archiveDependencyGraphFile.gml.
classDependencyGraphFile File - If specified, the plugin renders class dependency graph into this file.
Default value is: target/storyteller/classDependencyGraph.gml.
graphVizDotFile String - The plugin uses GraphViz package to render graphs in formats like PDF and so on. If the dot executable is not in PATH, it can be specified manually here.
Default value is: dot.
rootedArchiveDependencyGraphFile File - If specified, the plugin renders "rooted" archive dependency graph into this file.
Default value is: target/storyteller/rootedArchiveDependencyGraph.gml.
rootedClassDependencyGraphFile File - If specified, the plugin renders "rooted" class dependency graph into this file.
Default value is: target/storyteller/rootedClassDependencyGraph.gml.

Parameter Details

archiveDependencyGraphFile:

If specified, the plugin renders archive dependency graph into this file.
  • Type: java.io.File
  • Required: No
  • Expression: ${archiveDependencyGraphFile}
  • Default: target/storyteller/archiveDependencyGraphFile.gml

classDependencyGraphFile:

If specified, the plugin renders class dependency graph into this file.
  • Type: java.io.File
  • Required: No
  • Expression: ${classDependencyGraphFile}
  • Default: target/storyteller/classDependencyGraph.gml

graphVizDotFile:

The plugin uses GraphViz package to render graphs in formats like PDF and so on. If the dot executable is not in PATH, it can be specified manually here.
  • Type: java.lang.String
  • Required: No
  • Expression: ${graphViz.dotFile}
  • Default: dot

rootedArchiveDependencyGraphFile:

If specified, the plugin renders "rooted" archive dependency graph into this file.
  • Type: java.io.File
  • Required: No
  • Expression: ${rootedArchiveDependencyGraphFile}
  • Default: target/storyteller/rootedArchiveDependencyGraph.gml

rootedClassDependencyGraphFile:

If specified, the plugin renders "rooted" class dependency graph into this file.
  • Type: java.io.File
  • Required: No
  • Expression: ${rootedClassDependencyGraphFile}
  • Default: target/storyteller/rootedClassDependencyGraph.gml
